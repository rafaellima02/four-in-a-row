# 🎮 Projeto Interdisciplinar — Jogo Baseado em Turnos

## 📚 Sobre o Projeto

Este projeto foi desenvolvido como atividade da disciplina de **Lógica de Programação** no Instituto Federal de Pernambuco (IFPE), no curso de **Tecnologia em Sistemas para Internet (TSI)**.

A proposta consiste na criação de um **jogo baseado em turnos**, utilizando conceitos fundamentais de programação para desenvolver mecânicas, regras e interações entre os jogadores.

Além da aplicação prática da lógica de programação, o projeto também integra conhecimentos de outras áreas do curso, tornando-se uma atividade interdisciplinar.

# 🕹️ Funcionamento do Jogo

O jogo será estruturado em um sistema de turnos, onde cada jogador terá sua vez para realizar uma jogada no tabuleiro.

Cada participante utilizará uma cor diferente:

* 🔴 Vermelho
* 🟡 Amarelo

O objetivo principal é formar uma sequência de **4 peças da mesma cor**, podendo ser:

* horizontal;
* vertical;
* ou diagonal.

O primeiro jogador que conseguir completar essa sequência vence a partida.

---

# ⚙️ Mecânica das Peças

O jogo possui uma mecânica baseada em gravidade. Isso significa que as peças não podem ficar “flutuando” no tabuleiro.

Ao selecionar uma coluna:

* a peça cairá automaticamente até a última posição disponível;
* para que uma peça permaneça em uma posição superior, é necessário que exista outra peça abaixo dela servindo de apoio.

Dessa forma, o jogador precisa pensar estrategicamente antes de cada jogada, analisando:

* os espaços disponíveis;
* as possibilidades do adversário;
* e as próximas combinações possíveis.

---

# 🧩 Estratégia do Jogo

O jogo exige estratégia e planejamento, pois os jogadores devem:

* criar oportunidades para formar sequências;
* bloquear as jogadas do adversário;
* prever movimentos futuros;
* aproveitar corretamente os espaços do tabuleiro.

Com isso, o projeto combina lógica, estratégia, design e programação em uma experiência interativa e educativa.
